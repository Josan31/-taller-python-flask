# Estructuras de Control — if, elif, else

## ¿Qué son las estructuras de control?

Las estructuras de control son instrucciones que determinan el flujo de ejecución de un programa

En otras palabras es la forma en que le decimos al programa: si pasa esta condición haz esto, de lo contrario toma esta otra decisión.

## ¿Qué son if, elif y else y cómo funcionan?

Estas son las estructuras de control básicas de python y su funcionamiento es el siguiente:

- **if**: Es la estructura básica que permite evaluar una condición. Si la condición es verdadera, se ejecuta el código que está dentro del if.

- **elif**: Es una estructura que nos permite seguir evaluando condiciones si la condición del if o del elif anterior es falsa.

- **else**: Es la estructura que se ejecuta si la condición del if o del elif anterior es falsa. suele usarse como última opcion si todo lo demás falla.

## Identación

La identación al hacer estas condiciones es esencial, ya que de no cumplirlas el programa no entenderá qué código pertenece a cada estructura.
