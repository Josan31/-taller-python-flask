# Listas, Tuplas y Diccionarios

Las Listas, Tuplas y Diccionarios son estructuras de datos nativas de Python que se utilizan para almacenar colecciones de elementos bajo un solo nombre

En palabras mas simples, son "cajas" en las que podemos guardar diferentes tipos de valores, siguiendo un concpeto similar al de las variables.

La diferencia es que estas pueden almacenar mas de un valor.

Por ejemplo, una **lista** puede almacenar numeros, textos, booleanos, etc. En esta se usan los **corchetes []**

Las **tuplas** son similares a las listas, pero no se pueden modificar. En esta se usan los **parentesis ()**

Los **diccionarios** son similares a las listas, pero en lugar de almacenar valores, almacenan pares de clave-valor. En esta se usan las **llaves {}**

## ¿Cómo se recorren?

Las listas y las tuplas se recorren exactamente igual porque ambas son colecciones ordenadas de elementos secuenciales, es decir, que cada elemento tiene una posicion

los diccionarios tiene 3 formas distintas de recorrerse

1. Recorrer solo las claves
2. Recorrer Recorrer solo los Valores (Usando .values())
3. Recorrer Recorrer Llaves y Valores a la vez (Usando .items())
