# Variables y Tipos de Datos

## ¿Qué es una Variable?

Una variable es un contenedor o espacio en la memoria de la computadora (RAM) donde se almacena un dato.

En palabras mas simples es como una caja donde guardamos informacion para poder usarla mas adelante.

## ¿Qué son los Tipos de Datos y cuáles son?

Los tipos de datos son los diferentes tipos de informacion que se pueden almacenar en una variable. Entre estos se encuentran:

- **Enteros (int)**: Numeros sin decimales.
- **Flotantes (float)**: Numeros con decimales.
- **Cadenas de texto (str)**: Texto que va entre comillas.
- **Booleanos (bool)**: Valores True o False.

## ¿Cómo se declara una variable?

En python una variable se declara usando el signo igual (=) para asignar un valor a un nombre de variable. Ejemplo:

**nombre = "Carla"**

Siendo el nombre (identificador) de la variable **nombre** y el valor **"Carla"**.

## ¿Qué es el tipado dinámico?

El tipado dinámico es una característica muy esencial de este lenguaje que permite que no sea necesario declarar el tipo de variable al crearla. Además, esto permite que, a lo largo del código, una misma variable pueda cambiar de tipo de dato si se le asigna un valor diferente sin generar errores.
