# Manejo de Excepciones — try, except, finally

Los bloques try / except permiten gestionar errores en tiempo de ejecución para evitar que tus programas colapsen.

Son como capturadores de errores, que evita que el programa simeplemente se detenga si mostrar nada. Existen tres importantes que son:

- **try**: Aloja el código que podría fallar.
- **except**: Se activa solo si ocurre un error, ejecutando una alternativa.
- **finally**: Ejecuta un bloque de código si o si al finalizar, haya un error o no.
- **ValueError**: Este es un error que ocurre cuando se intenta convertir un valor que no es válido.
- **ZeroDivisionError**: Este es un error que ocurre cuando se intenta dividir un número por cero.
