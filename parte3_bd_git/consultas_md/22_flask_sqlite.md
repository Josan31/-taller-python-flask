# Integracion Flask + SQLite — CRUD basico

El acrónimo CRUD representa las cuatro operaciones fundamentales que se pueden realizar en cualquier base de datos: **Create** (Crear), **Read** (Leer), **Update** (Actualizar) y **Delete** (Eliminar).

## ¿Cómo conectar Flask con SQLite?

Para conectar Flask con SQLite, se debe diseñar un mecanismo que permita a las rutas del servidor comunicarse con el archivo de la base de datos.
