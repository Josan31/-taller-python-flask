# ¿Qué es Flask y como instalarlo?

es un popular **micro-framework** para desarrollo web escrito en Python.
Se le llama **"micro-framework"** porque incluye las herramientas mínimas necesarias para poner en marcha una aplicación web o API, lo que le otorga una gran ligereza y flexibilidad.

## Diferencias entre un framework y un microframework

La diferencia radica en que un **framework** son herramientas que ya vienen fabricadas dentro del código, ayuda a evitar empezar desde cero con codigo reutilizable.

Un **microframework** es es mas sencillo y ligero incluyendo solo lo esencial y el resto de funcionalidades debes de instaalrlas con extensiones.

## ¿Cuáles son las diferencias con Django?

La mayor diferencia es que **Django** es un **Framework** por ende es mucho mas completo que **flask**. Trayendo herramientas complejas y código reutilizable por defecto para agilizar el desarrollo.

## Instalación de Flask

Para instalar Flask, debemos de ejecutar el siguiente comando en la terminal:

**pip install flask**
